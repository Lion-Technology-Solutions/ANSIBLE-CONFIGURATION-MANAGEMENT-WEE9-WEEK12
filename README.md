# Week 9 - Ansible

This project contains Ansible playbooks and Terraform configurations for infrastructure automation.

## Structure

- `terraform/`: Terraform configurations for provisioning infrastructure.

## Getting Started

1. Ensure you have Ansible and Terraform installed.
2. Configure your inventory in `terraform/inventory.ini`.
3. Run Terraform commands in the `terraform/` directory.

## Usage

- To apply Terraform changes: `cd terraform && terraform apply`
- To run Ansible playbooks: `ansible-playbook playbook.yml`
